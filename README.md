# 🧮 Simple Calculator in C++

A console-based calculator application developed using **C++** that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

This project also supports **continuous calculations**, allowing users to perform multiple operations without restarting the program.

---

# 📌 Project Description

The calculator allows users to:

- Enter two numbers
- Choose an arithmetic operation
- View the calculated result
- Continue performing operations using the previous result

The project demonstrates the use of:
- Variables
- User Input/Output
- Conditional Statements
- Switch Case
- Loops
- Arithmetic Operations

# ⚙️ How the Program Works

## Step 1
The user enters:
- First number
- Second number

## Step 2
The user selects an operation:
```text
+  Addition
-  Subtraction
*  Multiplication
/  Division
```

## Step 3
The program performs the selected calculation and displays the result.

## Step 4
The user can continue calculations using the previous result.

## Step 5
The program ends when the user chooses to exit.

---

# ▶️ How to Run the Program

## Compile the Program

```bash
g++ main.cpp -o calculator
```

## Run the Program

```bash
./calculator
```
